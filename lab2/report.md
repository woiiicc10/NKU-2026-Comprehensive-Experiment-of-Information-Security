### 课时一 DDOS攻击和ARP欺骗

#### 一、DDOS攻击的基本概念

### 1.1 什么是DDOS攻击

DDOS（Distributed Denial of Service，分布式拒绝服务）攻击是一种常见的网络攻击方式：
- 攻击者利用多个攻击源同时向目标服务器发送大量请求
- 目标服务器资源耗尽，无法正常提供服务
- 合法用户无法访问，服务中断

### 1.2 DDOS攻击特点

- 攻击流量来源分散
- 难以追踪和阻断
- 对带宽、CPU、内存压力大

## 二、LOIC工具的安装与配置

LOIC（Low Orbit Ion Cannon）是用于模拟DDOS攻击的工具：

- 下载地址：公开GitHub或测试网络资源
- 配置参数：目标IP、目标端口、线程数、协议（TCP/UDP/HTTP）

## 四、ARP欺骗

### 4.1 什么是ARP欺骗

ARP欺骗是一种中间人攻击，通过伪造ARP应答让受害者的ARP缓存映射到攻击者。

### 4.2 dsniff安装与使用

- 安装：sudo apt-get install dsniff
- 工具：arpspoof、dnsspoof、urlsnarf等

### 4.3 启用IP转发

```bash
echo 1 > /proc/sys/net/ipv4/ip_forward
```

### 4.4 发起ARP欺骗

```bash
arpspoof -i eth0 -t 目标IP -r 网关IP
```

## 三、sqlmap使用

### 3.1 安装（从官网下载）

1. 访问sqlmap官网：https://sqlmap.org/ 或 https://github.com/sqlmapproject/sqlmap
2. 下载最新稳定压缩包（zip/tar.gz）
3. 解压到本地目录
4. 运行：
```
cd sqlmap
python sqlmap.py -h
```

（本次实验为官网下载安装，未使用git克隆）

### 3.2 注入测试

- 获取DB：--dbs
- 获取表：-D 数据库 --tables
- 获取列：-D 数据库 -T 表 --columns
- dump数据：--dump

## 四、安全防御总结

1. SQL注入：使用参数化查询、输入过滤、WAF
2. Cookie攻击：HttpOnly/Secure、同源策略、会话超时
3. 文件包含：禁止远程包含、路径白名单、最小权限
4. 文件上传：内容检测、存储隔离、禁止执行
5. DDoS/ARP：启用防火墙、网络隔离、ARP检测
